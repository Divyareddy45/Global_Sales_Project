**üóÇ DATA MODEL (STAR SCHEMA)**
The project follows a Star Schema for optimized performance and clean analytics.

FACT TABLES
- Fact_Sales     : Sales transaction data
- Fact_Returns   : Product return information

DIMENSION TABLES
- DimDate        : Date hierarchy (Year, Month, MonthYear)
- DimCustomer    : Customer details
- DimProduct     : Product, Category, Sub-category
- DimRegion      : Geographic regions
- DimPerson      : Sales representatives
